# To install

```
python3 -m venv venv
act
pip install --upgrade pip
pip install textual kinneyotp

python otp
```

# Tips:
- How to select text to copy:
  * iTerm Hold the OPTION key.
  * Gnome Terminal Hold the SHIFT key.
  * Windows Terminal Hold the SHIFT key.
